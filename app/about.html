RaspberryPI 2

Software used to capture and download images onto RaspberryPI is gPhoto2. Only older consumer cameras support gPhoto2 remote control over usb - gPhoto2 Remote control. New prosumer and pro digital SLR cameras are more fully supported for remote control. I run the script below every 2 minutes on the RaspberryPi and then copy it to my server which then creates a movie from the single frames every hour. 

#!/bin/bash
# SamArriola
# 5-22-2015
#
# Capture and download the image. Use ImageMagick utility to write the date and
# time in top right corner. Then copy the file over to home linux server.
cd /home/pi/cam/
rm 2min.jpg
/usr/local/bin/gphoto2 --capture-image-and-download --filename 2min.jpg
fdate=$(date "+%b_%d_%Y_%H_%M")
adate=$(date "+%b %d, %Y:%H:%M ")
convert 2min.jpg -gravity northeast -pointsize 40 \
    -stroke black -strokewidth 4 -annotate 0 "${adate}" \
    -stroke white -strokewidth 1 -fill white -annotate 0 "${adate}" \
     "${fdate}".jpg
scp "${fdate}".jpg sarriola@plex:/var/www/cam/
